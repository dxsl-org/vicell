## Step 0: Add `Computed::into_parts()`

In `libs/viui/src/signal.rs`, add to `impl<T: 'static> Computed<T>`:

```rust
/// Decompose into the inner signal and the subscription handle.
///
/// The returned `SubscriptionHandle` must be stored to keep automatic
/// updates active — dropping it stops the computed from tracking.
pub fn into_parts(self) -> (Signal<T>, SubscriptionHandle) {
    (self.signal, self._handle)
}
```

This is a non-breaking addition (new method, not in `libs/api/`).
Verify: `cargo check -p viui` passes.

## CodeGen Struct

```rust
pub struct CodeGen {
    // Counter for generating unique variable names
    clone_counter: usize,
    sub_counter:   usize,
    widget_counter: usize,
}

impl CodeGen {
    pub fn new() -> Self { ... }
    pub fn generate(&mut self, file: &ViFile) -> String { ... }
}
```

`generate()` dispatches to `gen_component()` for each component. Joins with `\n`.

---

## Code Emission Strategy

Build the output as a `Vec<String>` of lines, joined at the end. Each function returns the lines it owns; outer functions `extend` their buffer.

No string template engine — just `push_str` / `format!` on a `String` builder inside each helper.

## Generated File Structure

```rust
// Generated by vi-compiler — DO NOT EDIT
#![allow(unused_variables, dead_code)]
extern crate alloc;

use alloc::boxed::Box;
// (only imports for types actually used by this component)
use viui::signal::{Signal, SubscriptionHandle};
use viui::node::ViNode;
use viui::node_widgets::label::Label;       // if Text/Label used
use viui::node_widgets::button::Button;     // if Button used
use viui::node_widgets::column::Column;     // if VerticalLayout/VBox used
use viui::canvas::Color;                    // if any ColorLit used

/// Generated component: {ComponentName}
pub struct {ComponentName} {
    pub {prop_name}: Signal<{RustType}>,
    // ... subscription handles for each interpolated binding
    _{sub_name}: SubscriptionHandle,
}

impl {ComponentName} {
    pub fn build() -> (Self, {RootWidgetType}) {
        // 1. Signal declarations (from properties)
        // 2. Derived text signals (from Interpolated bindings)
        // 3. Widget instantiation (leaf-first, depth-first)
        // 4. Layout container
        // 5. Return (Self { ... }, layout)
    }
}
```

---

## gen_component() Logic

```
Input: &Component
Output: (component_struct_lines, impl_lines)
```

1. **Collect imports needed** — set of element types used (walk tree)
2. **Emit struct fields**:
   - For each `PropertyDecl` with `visibility = Some(InOut | Out)`: `pub {name}: Signal<{rust_type}>`
   - For each interpolated text binding found in element tree: `_{sub_name}: SubscriptionHandle`
3. **Emit `build()` body**:
   a. Signal declarations from properties (with default values)
   b. Walk element tree → collect `(stmts, widget_var_name, sub_handle_names)`
   c. Emit root widget (outermost element becomes return type)
   d. Emit `Self { fields }` and `(state, root_widget)`

---

## gen_element() Logic (recursive)

```
Input: &Element, element_index: usize
Output: (pre_stmts: Vec<String>, widget_expr: String, sub_handles: Vec<String>)
```

- `pre_stmts`: lines to emit BEFORE the widget variable declaration
- `widget_expr`: Rust expression creating the widget (e.g. `Label::new(text_0).with_color(...)`)
- `sub_handles`: names of `SubscriptionHandle` variables created (must be stored in struct)

### Container elements (VerticalLayout, HorizontalLayout)

```rust
// For each child element recursively
let (child_stmts, child_var, child_subs) = gen_element(child, idx);
// ... accumulate children

// Emit Column::new(alloc::vec![Box::new(w_x) as Box<dyn ViNode>, ...])
// Apply bindings: padding → .with_padding(f), spacing → .with_spacing(f)
```

### Text element

```rust
// text binding: eval_binding(raw, "text", "Text")
// - Interpolated → emit derived signal + subscribe; add sub_handle to list
// - StringLit → Signal::new(raw.to_string())
// color binding → .with_color(Color::rgb(r, g, b)) or Color::WHITE
```

### Button element

```rust
// text binding → &str literal (no Signal needed for Button label)
// clicked callback → emit closure body: eval_callback(body)
// - AugAssign { name, Add, IntLit(n) } → {name}_cn.update(|x| *x += n)
// Clone source signal into closure: let {name}_cn = {name}.clone();
```

## Variable Naming

| Purpose | Pattern |
|---------|---------|
| Derived text signal | `text_{n}` where n = `sub_counter` |
| Subscription handle | `_sub_text_{n}` |
| Signal clone for closure | `_{sig_name}_c{n}` where n = `clone_counter` |
| Widget variable | `w_{element_type_lower}_{n}` |

All counters increment per-usage within a component (reset per component).
